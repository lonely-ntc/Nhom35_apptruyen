```

    вҶ“

    вҶ“

    вҶ“

### 2. Dashboard Statistics Flow
```
Dashboard Init
    вҶ“
Parallel Fetch:
    в”ңв”ҖвҶ’ Firestore.collection('users').get()
    в”ңв”ҖвҶ’ DatabaseService.getStories()
    в””в”ҖвҶ’ Firestore.collection('stories/{id}/comments').get()
    вҶ“

    вҶ“

### 3. Story Image Loading Flow
```
Story Card Build
    вҶ“
FutureBuilder<String>
    вҶ“
ImageHelper.getImageFromStory()
    в”ңв”ҖвҶ’ Check if URL (http/https)
    в”ңв”ҖвҶ’ Check assets path
    в””в”ҖвҶ’ Normalize path
    вҶ“
Return image path
    вҶ“
Image Widget
    в”ңв”ҖвҶ’ NetworkImage (if URL)
    в””в”ҖвҶ’ AssetImage (if local)
    вҶ“
Error? вҶ’ Fallback Placeholder
```

### 4. User Management Flow
```
User Screen Init
    вҶ“
StreamBuilder<QuerySnapshot>
    вҶ“
Firestore.collection('users').snapshots()
    вҶ“
Filter by Search Query
    вҶ“
Sort (Admin first)
    вҶ“
Build User Cards
    вҶ“
User Action (Toggle Admin)
    вҶ“
Validate (not self)
    вҶ“
Update Firestore
    вҶ“
Realtime Update UI
```

  return error("Cannot change own permissions");

  return error("Permission denied");

  return redirect(MainScreen);
